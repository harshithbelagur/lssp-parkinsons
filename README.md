# ELEN 6889 - Large Scale Stream Processing with Prof. Deepak Turaga at Columbia University

This repo is as part of the final course project for the aforementioned course. We used GCP for deploying all the components and made use of Dataproc, Pub/Sub, Datastore, and Cloud Functions. The dataset used can be found at https://www.physionet.org/content/parkinsons-disease-smartwatch/1.0.0/
